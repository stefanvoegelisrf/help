## map

To convert a value from a a specified source range into a value at the same position of a destination range, we can use ```map```.

```
map(
    value,
    sourceRangeStart,
    sourceRangeEnd,
    destinationRangeStart,
    destinationRangeEnd
)
```

```
map()
```
